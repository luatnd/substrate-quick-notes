Common

Check cargo err

cargo check -p node-template-runtime

You can use IDE feature instead, eg: Jetbrain auto detect cargo.

Build pallet only

cargo build -p pallet-kitties

Build release

cargo build --release

It will rebuild only new cargo.
If rust / cargo version was changed => Rebuild all

Start node

./target/release/node-template --dev --tmp

Many options here: ./target/release/node-template --help

Frequent options:

  • --tmp
